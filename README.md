# single-page-application-spa-wordly

Summary
This is a Single Page Application (SPA) I built that lets you look up definitions for English words. 
I made it to practice working with APIs, loops, and building things on the page using JavaScript.

Installastion
- npm install

How to use:
- Clone from git hub using got clone
- Find folder and open index.html using explorer.exe or open

Languages Used:
- HTML/CSS
- JavaScript

Future updates:
- Adding local storage for users
